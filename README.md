# Jellyfin-Milkdrop2

this adds a button for visualiser that uses the butterchrun implementation of milkdrop2 visualiser 

a massive thanks to jberg visit the original butterchurn repo here https://github.com/jberg/butterchurn 

## demo video:

https://www.blueboxofdoom.uk/filebrowser/api/public/dl/naz24nh-?inline=true

## installation:
put milkdrop.js into your webroot `/usr/share/jellyfin/web/` 

edit index.html and add `<script defer src="milkdrop.js"></script>` just after `</style>` and save it


## keybinds 
`N` next random preset
`A` Auto cycle preset
`Escape` close visualiser 
`left arrow` decrease sensitivity 
`right arrow` increase sensitivity

## Known bugs
it may or may not work on your mobile device its a ymmv but it does work on pc platforms in multiple browsers
